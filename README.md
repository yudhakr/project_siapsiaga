# 🚨 SIAPSIAGA  
### Disaster Preparedness System with AI Analysis

SIAPSIAGA adalah aplikasi pelaporan bencana berbasis AI yang memungkinkan pengguna untuk:

- 📸 Upload foto kejadian
- 🤖 Analisis gambar otomatis menggunakan AI
- 📍 Input lokasi kejadian
- 📊 Melihat laporan terbaru
- 🛡 Panel admin monitoring

Project ini menggunakan:

- ⚛ React + Vite + TypeScript  
- 🎨 Tailwind CSS  
- ☁ Supabase (Database + Storage + Edge Function)  
- 🧠 AI Vision (melalui Supabase Edge Function)  

---

# 🛠️ Requirements

Pastikan sudah menginstall:

- Node.js ≥ 18
- npm / pnpm / yarn
- Supabase CLI
